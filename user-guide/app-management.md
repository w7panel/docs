# 应用管理

## 应用列表

应用列表显示所有已部署的应用，包括应用名称、站点域名、创建时间等信息。

### 创建应用

1. 点击「新建」按钮
2. 选择创建方式：
   - **应用商店**: 从模板快速部署
   - **Docker Compose**: 使用 compose 文件部署
   - **K8s Yaml**: 使用 Kubernetes 配置部署
   - **代码包**: 上传代码包部署
   - **Helm**: 使用 Helm Chart 部署

### 应用操作

每个应用提供以下操作：

| 操作 | 说明 |
|------|------|
| 域名管理 | 配置应用的访问域名 |
| 文件管理 | 管理应用文件系统 |
| 删除 | 删除应用 |

## 文件管理

点击「文件管理」进入应用的文件系统。

### 文件操作

| 操作 | 方法 |
|------|------|
| 上传文件 | 点击「上传文件」按钮 |
| 新建文件/目录 | 点击「新建」→ 选择类型 |
| 下载文件/目录 | 点击「复制」按钮 |
| 删除文件/目录 | 勾选后点击「删除」按钮 |
| 重命名 | 右键点击文件 → 重命名 |
| 修改权限 | 勾选后点击「权限」按钮 |

### 压缩与解压

**压缩文件:**
1. 勾选要压缩的文件/目录
2. 点击「压缩」按钮
3. 选择压缩格式:
   - zip（通用格式）
   - tar（无压缩）
   - tar.gz（gzip压缩）
   - tar.xz（xz压缩）
4. 设置压缩路径和文件名
5. 点击「压缩」按钮

**解压文件:**
1. 找到压缩文件
2. 点击解压图标
3. 选择解压目标路径
4. 点击「确定」

> 支持的解压格式: zip, tar, tar.gz, tar.bz2, tar.xz, 7z

### 开发编辑器

点击「开发编辑器」可以在线编辑代码文件，支持：
- 语法高亮
- 文件树浏览
- 代码编辑
- 实时保存

## 域名管理

### 添加域名

1. 点击「域名管理」
2. 点击「添加域名」
3. 输入域名（如: example.com）
4. 选择是否启用 HTTPS
5. 点击「确定」

### HTTPS 证书

如果启用 HTTPS，需要：
1. 上传 SSL 证书
2. 或使用 Let's Encrypt 自动申请

## 计划任务

计划任务用于定时执行脚本。

### 创建计划任务

1. 进入「计划任务」
2. 点击「新建」
3. 配置：
   - 任务名称
   - 执行时间（Cron 表达式）
   - 执行脚本
4. 点击「确定」

### Cron 表达式

```
* * * * *
│ │ │ │ │
│ │ │ │ └─ 星期几 (0-6, 0=周日)
│ │ │ └─── 月份 (1-12)
│ │ └───── 日期 (1-31)
│ └─────── 小时 (0-23)
└───────── 分钟 (0-59)
```

示例:
- `0 * * * *` - 每小时执行
- `0 0 * * *` - 每天零点执行
- `*/5 * * * *` - 每5分钟执行

## 反向代理

反向代理用于将外部请求转发到内部服务。

### 添加代理

1. 进入「反向代理」
2. 点击「新建」
3. 配置：
   - 域名
   - 目标地址
   - 路径规则
4. 点击「确定」
