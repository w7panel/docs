# 文件管理

文件管理模块提供对应用容器内文件系统的完整操作能力。

## 进入文件管理

1. 进入「应用管理」→「应用列表」
2. 找到目标应用
3. 点击「文件管理」

## 基本操作

### 上传文件

1. 点击「上传文件」按钮
2. 选择本地文件（支持多选）
3. 等待上传完成

> 支持拖拽上传

### 新建文件/目录

1. 点击「新建」→ 选择类型
2. 输入名称
3. 点击「确定」

### 下载文件

**单个文件:**
- 点击「复制」按钮下载

**目录:**
- 系统自动打包为 zip 下载

### 删除文件

1. 勾选要删除的文件/目录
2. 点击「删除」按钮
3. 确认删除

> ⚠️ 删除操作不可恢复，请谨慎操作

### 重命名

1. 右键点击文件
2. 选择「重命名」
3. 输入新名称
4. 按 Enter 确认

## 批量操作

### 选择多个文件

- 点击文件前的复选框
- 点击表头的「全选」选择所有文件

### 批量压缩

1. 勾选多个文件
2. 点击「压缩」按钮
3. 选择格式并设置路径
4. 点击「压缩」

### 批量删除

1. 勾选多个文件
2. 点击「删除」按钮
3. 确认删除

## 压缩与解压

### 压缩文件

1. 勾选要压缩的文件
2. 点击「压缩」按钮
3. 配置压缩选项:

| 选项 | 说明 |
|------|------|
| 压缩类型 | zip / tar / tar.gz / tar.xz |
| 压缩路径 | 压缩文件保存位置 |

4. 点击「压缩」

### 解压文件

1. 找到压缩文件
2. 点击解压图标（📤）
3. 配置解压选项:

| 选项 | 说明 |
|------|------|
| 解压路径 | 文件解压目标位置 |
| 转换GBK文件名 | 解决中文乱码问题 |

4. 点击「确定」

### 支持的格式

| 操作 | 支持格式 |
|------|---------|
| 压缩 | zip, tar, tar.gz, tar.xz |
| 解压 | zip, tar, tar.gz, tar.bz2, tar.xz, 7z |

## 权限管理

### 修改权限

1. 勾选文件/目录
2. 点击「权限」按钮
3. 设置权限:

| 权限 | 数值 | 说明 |
|------|------|------|
| 读取 | 4 | 可查看内容 |
| 写入 | 2 | 可修改内容 |
| 执行 | 1 | 可执行（程序/脚本） |

常见权限组合:
- 755: 所有者全部权限，其他人读取和执行
- 644: 所有者读写，其他人只读
- 777: 所有人全部权限（不推荐）

### 修改所有者

1. 在权限对话框中选择「所有者」
2. 输入用户名
3. 勾选「递归」应用到子目录
4. 点击「确定」

## 开发编辑器

### 打开编辑器

点击「开发编辑器」按钮，打开在线 IDE。

### 编辑器功能

- **文件树**: 左侧显示目录结构
- **代码编辑**: 支持语法高亮
- **多标签**: 同时编辑多个文件
- **自动保存**: 编辑后自动同步

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存文件 |
| Ctrl+P | 快速打开文件 |
| Ctrl+B | 切换侧边栏 |
| Ctrl+` | 打开终端 |

## 文件搜索

在文件列表上方可以搜索文件名：
1. 输入搜索关键词
2. 自动过滤匹配的文件

## 路径导航

点击路径导航可以快速跳转：
- 点击「根目录」返回根路径
- 点击任意路径段跳转到该位置

## 常见问题

### Q: 上传文件大小限制？
A: 默认限制 100MB，可在系统配置中调整。

### Q: 中文文件名乱码？
A: 解压时勾选「转换GBK文件名」选项。

### Q: 无法删除文件？
A: 检查文件权限，确保有写入权限。

### Q: 编辑器无法保存？
A: 刷新页面后重试，或检查网络连接。
