# W7Panel 用户操作手册

## 快速入门

### 登录面板

1. 打开浏览器，访问面板地址（如 http://your-server:8080）
2. 输入用户名和密码
3. 完成滑块验证（如已启用）
4. 点击「登录」按钮

> 默认账号: admin / 123456

### 界面概览

登录后进入面板首页，可以看到：

```
┌─────────────────────────────────────────────────────┐
│  概览                                               │
├─────────────────────────────────────────────────────┤
│  [集群终端]                                         │
│                                                     │
│  节点 1个    应用 13个    域名地址 11个    存储分区 6个 │
│                                                     │
│  系统信息                                           │
│  集群版本: v1.31.4+k3s1                             │
│  创建时间: 2024-01-01                               │
│  集群模式: 单机（内嵌sqlite）                        │
│                                                     │
│  16.88% CPU分配率    26.9% 内存分配率               │
│  已分配/总量        已分配/总量                     │
└─────────────────────────────────────────────────────┘
```

- **集群概览**: CPU、内存、硬盘使用率实时图表
- **节点信息**: 集群节点数量和状态
- **应用列表**: 已部署的应用数量
- **域名地址**: 已配置的域名数量
- **存储分区**: 存储使用情况

## 功能模块

### 左侧菜单

| 菜单 | 子菜单 | 功能 |
|------|--------|------|
| **集群管理** | 概览 | 集群资源使用率监控、系统信息 |
| | 节点管理 | 节点注册、镜像源配置、内存优化、节点操作 |
| | 资源对象浏览器 | 浏览所有 K8S 资源对象 |
| **应用管理** | 应用列表 | 应用部署、域名管理、文件管理 |
| | 计划任务 | CronJob 定时任务 |
| | 反向代理 | Ingress 代理规则配置 |
| | 集群数据库 | 数据库服务（MySQL/PostgreSQL/Redis） |
| | AI 应用管理 | GPUStack GPU 应用部署 |
| **存储管理** | 存储设备 | 节点磁盘监控、I/O 统计 |
| | 存储分区 | PVC 创建、扩容、快照 |
| **制品管理** | - | ZPK 制品安装、镜像构建 |
| **系统管理** | 用户管理 | 用户、用户组、权限 |
| | 系统配置 | 云配置、API密钥、许可证 |
| | 订单/费用 | 订单管理、费用统计 |

### 应用部署方式

| 方式 | 说明 |
|------|------|

| Helm Chart | Helm 应用部署 |
| Docker Compose | Compose 文件转换部署 |
| K8S YAML | 直接提交 YAML 配置 |
